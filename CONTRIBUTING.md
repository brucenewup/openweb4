# Contributing to OpenWeb4

[中文说明](#中文说明) · [English](#english)

---

## 中文说明

感谢你愿意参与改进 OpenWeb4。请先阅读 [README.md](README.md) / [README.en.md](README.en.md) 与 [SECURITY.md](SECURITY.md)。

- **Issue / 讨论**：描述复现步骤、环境（Java 版本、OS）、期望与实际行为。
- **Pull Request**：尽量单一主题、附带或更新相关测试；不要提交密钥、`.env` 或仅本地使用的配置。
- **代码风格**：与现有 Java / 前端代码保持一致；避免无关大重构。
- **许可**：提交即表示你同意以 **MIT** 授权你的贡献（与仓库 [LICENSE](LICENSE) 一致）。

## English

Thanks for helping improve OpenWeb4. Please read [README.en.md](README.en.md) and [SECURITY.md](SECURITY.md) first.

### Issues

- Describe steps to reproduce, environment (Java version, OS), expected vs actual behavior.
- For security-sensitive reports, use [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories) or follow [SECURITY.md](SECURITY.md); do not file public issues with live secrets.

### Pull requests

- Keep changes focused; update or add tests when behavior changes.
- Do not commit API keys, tokens, `.env` files, or machine-specific secrets.
- Match existing code style; avoid unrelated refactors in the same PR.
